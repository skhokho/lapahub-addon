# LapaHub Home Assistant Add-ons

This repository contains Home Assistant add-ons for LapaHub smart home energy management.

## Add-ons

### [LapaHub Bridge](./lapahub)

Connect your Home Assistant to LapaHub cloud for:
- Remote device control from the LapaHub mobile app
- AI-powered energy optimization
- Load shedding automation
- Energy monitoring and reporting

## Installation

1. Open your Home Assistant instance
2. Go to **Settings** → **Add-ons** → **Add-on Store**
3. Click the **⋮** menu (top right) → **Repositories**
4. Add this repository URL:
   ```
   https://github.com/skhokho/lapahub-addon
   ```
5. Click **Add** → **Close**
6. Refresh the page and find **LapaHub Bridge** in the store
7. Click **Install**

## Requirements

- Home Assistant OS or Supervised installation
- LapaHub account with registered hub
- Active internet connection
